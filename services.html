Pengertian Website?

Website adalah kumpulan halaman yang berisi informasi tertentu dan dapat diakses dengan mudah oleh siapapun, kapanpun, dan di manapun melalui internet.

Jenis Jenis Website

Sebelum benar benar memesan website, alangkah baiknya jika mengenal jenis jenis website terlebih dahulu, sehingga tidak keliru ketika memesan tipe website yang diinginkan.
Jenis Penjelasan
Company Profile Website dengan konsep memperkenalkan perusahaan (profil perusahaan). Bisa memiliki database dan bisa pula tidak. Contoh :
  • https://lazaaq.github.io/projects/kelapa/index.html
  • https://lazaaq.github.io/projects/forklift/index.html
  • https://lazaaq.github.io/projects/sigma/index.html
  • https://lazaaq.github.io/projects/buana/index.html
E-Commerce Website dengan konsep katalog toko (berisi produk). Memilki Database. Bisa include dengan payment dan bisa pula tidak. Contoh :
  • https://alhijaztegal.com/
  • https://dekornata.com/Include Payment
  • https://www.satubaju.com/
Custom System Website dengan konsep custom. Pelanggan dapat dengan detail menjelaskan website yang ingin dibuat seperti apa. Bisa memiliki database. Contoh :
  • Sistem Kesehatan dan Keselamatan Rumah Sakit
  • Sistem Penerimaan Peserta Didik Baru Sekolah Link
  • Sistem Manajemen Kasir
  • Database Alumni Sekolah
  • Rencana Program dan Kegiatan Pembelajaran Semester (RPKPS)
