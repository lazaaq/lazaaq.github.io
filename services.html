Definition of Website?

Website is a collection of pages that contain certain information and can be accessed easily by anyone, anytime and anywhere via the internet.

Types of Website Types

Before actually ordering a website, it would be nice if you knew the types of websites first, so you weren't mistaken when ordering the type of website you wanted.
Type Explanation
Company Profile Website with the concept of introducing the company (company profile). Can have a database and can also not. Example :
  • https://lazaaq.github.io/projects/kelapa/index.html
  • https://lazaaq.github.io/projects/forklift/index.html
  • https://lazaaq.github.io/projects/sigma/index.html
  • https://lazaaq.github.io/projects/buana/index.html
E-Commerce Website with a store catalog concept (containing products). Own Databases. It can include payment or not. Example :
  • https://alhijaztegal.com/
  • https://dekornata.com/Include Payment
  • https://www.satubaju.com/
Custom System Website with custom concept. Customers can explain in detail what kind of website they want to make. Can have a database. Example :
  • Hospital Health and Safety System
  • New School Student Acceptance System Link
  • Cashier Management System
  • School Alumni Database
  • Semester Learning Program and Activity Plan - University
